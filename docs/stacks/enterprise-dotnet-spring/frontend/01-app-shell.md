# Frontend 01 - App Shell

## Yang Dibuat

Shell aplikasi enterprise untuk Angular atau React.

## Alur

1. Buat routing utama.
2. Buat layout login.
3. Buat layout dashboard.
4. Buat sidebar dan topbar.
5. Buat placeholder organization switcher.
6. Buat halaman not found dan forbidden.

## Output

Frontend punya struktur aplikasi operasional, bukan landing page.
