# Frontend 03 - Admin Dashboard

## Yang Dibuat

Dashboard enterprise untuk ringkasan operasional.

## Alur

1. Load organization aktif.
2. Load summary task/order.
3. Load recent activity.
4. Load error atau notification penting.
5. Tambahkan filter tanggal.

## Output

User bisa melihat kondisi aplikasi bisnis dari satu halaman.
