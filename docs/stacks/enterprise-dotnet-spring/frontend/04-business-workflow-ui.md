# Frontend 04 - Business Workflow UI

## Yang Dibuat

UI workflow utama: task atau order.

## Alur Task

1. Buat task list.
2. Buat create/edit form.
3. Buat task detail.
4. Buat status transition.
5. Buat assignment.

## Alur Order

1. Buat product list admin.
2. Buat create/edit product.
3. Buat order list.
4. Buat order detail.
5. Buat payment/fulfillment status.

## Output

Frontend menjalankan proses bisnis utama dari awal sampai akhir.
