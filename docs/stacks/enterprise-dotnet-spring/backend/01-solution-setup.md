# Backend 01 - Solution Setup

## Yang Dibuat

Fondasi backend enterprise dengan .NET atau Spring Boot.

## Alur .NET

1. Buat solution.
2. Buat project `Web`.
3. Buat folder/project `Modules`.
4. Tambahkan `SharedKernel`.
5. Tambahkan EF Core.
6. Tambahkan logging dan validation.
7. Tambahkan test project.

## Alur Spring Boot

1. Buat project Spring Boot.
2. Tambahkan Spring Web.
3. Tambahkan Spring Data JPA.
4. Tambahkan Spring Security.
5. Tambahkan Flyway atau Liquibase.
6. Tambahkan package `modules`.
7. Tambahkan test.

## Output

Backend siap menerima module domain tanpa mencampur controller, ORM, dan aturan bisnis.
