# Backend 07 - Observability Dan Deployment

## Yang Dibuat

Kesiapan production.

## Alur

1. Tambahkan structured logging.
2. Tambahkan correlation id.
3. Tambahkan health check `/health`.
4. Tambahkan readiness check `/ready`.
5. Tambahkan metrics.
6. Tambahkan tracing database dan external provider.
7. Buat Dockerfile.
8. Jalankan migration production.
9. Deploy ke VM, container platform, atau Kubernetes.

## Output

Backend bisa dipantau, di-debug, dan dioperasikan secara serius.
