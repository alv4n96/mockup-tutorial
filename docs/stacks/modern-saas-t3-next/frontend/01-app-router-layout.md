# Frontend 01 - App Router Layout

## Yang Dibuat

Struktur halaman Next.js untuk auth dan dashboard.

## Alur

1. Buat route group `(auth)`.
2. Buat route group `(dashboard)`.
3. Buat root layout.
4. Buat dashboard shell.
5. Buat sidebar, topbar, dan organization switcher placeholder.

## Struktur

```text
src/app/
  (auth)/
  (dashboard)/
    dashboard/
    tasks/
    settings/
```

## Output

Frontend punya struktur navigasi yang siap dihubungkan ke backend.
