# Frontend 03 - Dashboard Dan Organization

## Yang Dibuat

Dashboard awal dan pemilihan organization.

## Alur

1. Load current user.
2. Load organization list.
3. Jika belum ada organization, tampilkan create organization flow.
4. Simpan organization aktif di URL atau server state.
5. Load summary task berdasarkan organization aktif.

## Output

Dashboard hanya menampilkan data tenant yang benar.
