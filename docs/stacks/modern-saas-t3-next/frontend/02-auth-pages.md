# Frontend 02 - Auth Pages

## Yang Dibuat

Halaman sign in, sign up, dan session-aware redirect.

## Alur

1. Buat halaman `/sign-up`.
2. Buat halaman `/sign-in`.
3. Tambahkan form validation.
4. Submit ke auth provider atau tRPC mutation.
5. Jika berhasil, redirect ke `/dashboard`.
6. Jika session sudah ada, redirect dari auth page ke dashboard.

## Output

User bisa masuk ke aplikasi dan session tersedia untuk request dashboard.
