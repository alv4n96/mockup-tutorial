# Frontend 04 - Task UI Flow

## Yang Dibuat

UI CRUD task.

## Alur

1. Buat halaman `/tasks`.
2. Tambahkan filter status, assignee, dan search.
3. Buat drawer atau page `/tasks/new`.
4. Submit form create task.
5. Redirect ke detail task.
6. Buat update status.
7. Buat assign task.
8. Tampilkan empty, loading, dan error state.

## Output

User bisa membuat, membaca, dan mengubah task dari UI secara end-to-end.
