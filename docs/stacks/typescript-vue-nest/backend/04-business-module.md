# Backend 04 - Business Module

## Yang Dibuat

Task module atau catalog/order module.

## Alur Task

1. Buat project.
2. Buat task.
3. Buat endpoint create task.
4. Buat endpoint list task.
5. Buat endpoint update status.
6. Tambahkan tenant guard.

## Alur E-Commerce

1. Buat product.
2. Buat inventory.
3. Buat order.
4. Buat checkout endpoint.
5. Tambahkan transaksi.

## Output

Ada workflow bisnis nyata yang bisa dipakai frontend.
