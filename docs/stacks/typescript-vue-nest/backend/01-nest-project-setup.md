# Backend 01 - Nest Project Setup

## Yang Dibuat

Fondasi NestJS backend.

## Alur

1. Buat project NestJS.
2. Tambahkan PostgreSQL.
3. Tambahkan Prisma atau TypeORM.
4. Tambahkan config module.
5. Tambahkan validation pipe.
6. Tambahkan auth module placeholder.
7. Tambahkan test setup.

## Output

Backend Nest siap untuk module bisnis modular.
