# Frontend 02 - Auth Flow

## Yang Dibuat

Login dan route guard Vue.

## Alur

1. Buat halaman login.
2. Buat halaman register.
3. Simpan token/session sesuai backend.
4. Buat current user store.
5. Buat route guard.
6. Redirect user yang belum login.

## Output

Dashboard hanya bisa diakses user yang sudah login.
