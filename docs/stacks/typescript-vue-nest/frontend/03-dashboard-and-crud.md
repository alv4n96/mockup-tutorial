# Frontend 03 - Dashboard Dan CRUD

## Yang Dibuat

Dashboard dan CRUD modul utama.

## Alur

1. Load organization aktif.
2. Tampilkan summary.
3. Buat list task/product.
4. Buat create form.
5. Buat edit form.
6. Buat detail page.
7. Tampilkan loading, empty, dan error state.

## Output

Vue frontend bisa menjalankan workflow dari API sampai UI.
