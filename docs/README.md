## Cara Membaca Kode Implementasi

Untuk belajar dari nol dengan urutan coding yang diketik, mulai dari [tutorials/00-belajar-pemrograman-dari-nol.md](tutorials/00-belajar-pemrograman-dari-nol.md). Setelah itu lanjut ke [tutorials/14-step-by-step-programming-flow.md](tutorials/14-step-by-step-programming-flow.md), baca [stacks/shared/06-rbac-tenant-authorization.md](stacks/shared/06-rbac-tenant-authorization.md), lalu buka file `code-blueprint` di stack yang dipilih. Blueprint tersebut berisi contoh input request, role/permission RBAC, policy, use case, repository, controller/router, dan UI handling.
## Catatan

`docs/tutorials/` tetap dipertahankan sebagai referensi umum modular monolith lintas stack. Struktur utama pembelajaran tetap berada di `docs/stacks/`.
